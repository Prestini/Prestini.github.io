Figuras de Linguagem

Uma introdução às principais figuras de linguagem usadas na língua portuguesa

O que são Figuras de Linguagem?

Figuras de linguagem são recursos estilísticos usados para tornar a comunicação mais expressiva e envolvente. Elas podem ser usadas para destacar ideias, provocar emoções e gerar diferentes interpretações.

Principais Figuras de Linguagem

Metáfora

A metáfora é uma figura de linguagem que estabelece uma comparação entre dois conceitos sem o uso de termos que indiquem explicitamente a comparação. Esse recurso estilístico tem sido analisado há séculos e é amplamente utilizado tanto na linguagem falada quanto escrita.

Metonímia

Metonímia é uma figura de linguagem que consiste em substituir uma palavra ou termo por outro, mantendo a mesma mensagem. A palavra substituída e a que a substitui têm uma relação de proximidade de sentidos.

Hipérbole

Hipérbole é uma figura de linguagem que consiste em um exagero proposital para dar ênfase a algo. É uma figura de pensamento, ou seja, o sentido figurado reside na ideia sugerida.

Exemplo:

Antítese

A antítese é uma figura de linguagem que consiste na contraposição de palavras, expressões ou orações de sentidos opostos, com o objetivo de intensificar o sentido daquilo que se fala.

Exemplo:

Ironia

Ironia é uma figura de linguagem que consiste em dizer o oposto do que se quer expressar, ou sugerir algo diferente do que se afirma. É uma forma de expressão literária que pode ser usada para criticar, denunciar, censurar ou zombar de algo ou alguém.
